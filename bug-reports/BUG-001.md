# BUG-001: Unauthenticated User Can Access /inventory.html Without Login

| Field | Detail |
|---|---|
| **Bug ID** | BUG-001 |
| **Severity** | High |
| **Priority** | High |
| **Status** | Open |
| **Module** | Login / Security |
| **TC Reference** | TC_012 |
| **Found Date** | 06/06/2026 |
| **Environment** | Chrome 124 / Windows 11 |

## Steps to Reproduce
1. Open browser with no active session (not logged in)
2. Type `https://www.saucedemo.com/inventory.html` directly in address bar
3. Press Enter

## Expected Result
Browser redirects to `https://www.saucedemo.com/`
Login page is displayed.
Inventory page is NOT accessible without authentication.

## Actual Result
Inventory page loaded successfully without authentication.
No redirect occurred.
Unauthenticated user can view all products.

## Reference
NFR-S01 — Authentication Required
